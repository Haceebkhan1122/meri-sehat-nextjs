## 📋 Overview

This is the **Next.js frontend** for **Meri Sehat** — Pakistan's first AI-powered health platform. It serves as the main user-facing application handling:

- 🏠 **Landing page & marketing**
- 👤 **User authentication & dashboard**
- 🎥 **HD video consultations** with doctors
- ❤️ **rPPG vital sign tracking** via smartphone camera
- 📅 **Appointment booking & management**
- 📋 **Health records & prescriptions**
- 🌐 **Multilingual UI** (Urdu, English, Sindhi)
- ♿ **WCAG 2.1 AA compliant** accessibility

> **⚠️ Note:** This repository contains only the **Next.js frontend**. Backend API is separate.
